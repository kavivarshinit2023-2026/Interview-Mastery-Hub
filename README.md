Project Name: Interview Mastery Hub
Repo Link: https://github.com/kavivarshinit2023-2026/Interview-Mastery-Hub
Type: Frontend-only static web application (no backend server required)
Purpose: To help students, freshers, and early-career professionals prepare for job interviews by focusing on soft skills – the skills companies now check heavily in interviews (communication, teamwork, leadership, etc.).
Key Features (Professional Summary)

30-Day Soft Skills Challenge – Step-by-step daily plan to build better habits and skills
Topic-wise Learning Pages – Detailed explanations + curated video links for each skill
MCQ Practice – Multiple-choice quizzes to test understanding (one per skill)
Mock Interviews – Simulated practice questions and scenarios for each soft skill
AI-Powered Mock Interview – Basic interface for AI-style conversation practice
Resume Builder – Simple tool with fresher-friendly templates
User Authentication – Sign-up, login, personal dashboard (powered by Firebase)
Admin Dashboard – Basic panel to manage content (future-proof design)
Interview Tips Sections – Guides on dress code, body language, punctuality, smart questions, follow-up emails, etc.

Skills Covered

Communication
Teamwork
Leadership
Time Management
Emotional Intelligence
Problem Solving

Technical Details (Professional View)

Languages / Stack:
HTML → ~95% (38+ files)
CSS → Single global stylesheet (style.css)
JavaScript → Minimal (firebase.js + firebase-listeners.js for auth & data)

Architecture: Pure client-side → 100% static. Just open index.html in any browser – works offline after first load (except login).
Authentication: Firebase Email/Password (secure & easy integration)
Structure: Currently flat (no folders) – all 41 files in root directory for simplicity in early stage.
Deployment Ready: GitHub Pages, Netlify, Vercel, or local browser – zero server cost.
Current Stage: MVP (Minimum Viable Product) – single commit on Feb 14, 2026 → very fresh, clean start.

Strengths

Strong focus on soft skills (rare in most interview prep tools)
Fully offline-capable core features
Includes resume builder + mock practice → real portfolio value
Firebase already set up → easy to add user progress tracking later
